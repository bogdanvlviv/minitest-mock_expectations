# minitest-mock_expectations

Provides method call assertions.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'minitest-mock_expectations'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install minitest-mock_expectations
